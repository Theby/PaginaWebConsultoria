PaginaWebConsultoria
====================

Página web para una Empresa de Consultoría. De modo que sea posible manejar todo el sistema de negocio desde esta página.

Está página está desarrollada en:
- Ruby: 2.1.2
- Ruby on Rails: 4.1.1
